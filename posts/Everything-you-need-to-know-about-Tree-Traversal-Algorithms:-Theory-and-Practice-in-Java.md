<p>In computer science, a Tree is a widely used abstract data type (ADT), or data structure implementing this ADT, that simulates a hierarchical tree structure, with a root value and subtrees of children with a parent node, represented as a set of linked&#xA0;nodes.</p><figure><img alt src="https://hackernoon.com/hn-images/1*dlwiH2s79k2si49rEdd0wQ.png"></figure><p>A tree data structure can be defined recursively as a collection of nodes (starting at a root node), where each node is a data structure consisting of a value, together with a list of references to nodes (the &#x201C;children&#x201D;), with the constraints that no reference is duplicated, and none points to the&#xA0;root.</p><p><a href="https://en.wikipedia.org/wiki/Tree_(data_structure)"><em>Thanks to Wikipedia for that great definition of Tree data structure in computer&#xA0;science</em></a><em>.</em></p><figure><img alt src="https://hackernoon.com/hn-images/1*C0NTtEMcVbkA8xITf9TtKA.png"><figcaption>Tree sample</figcaption></figure><p>Trees can be traversed in several&#xA0;ways:</p><ul><li><strong>Pre Order Traversal</strong>: Root, Left, Right.<br>In our example: A B D H I E J C F G&#xA0;K</li><li><strong>In Order Traversal</strong>: Left, Root, Right. <br>In our example: H D I B J E A F C K&#xA0;G</li><li><strong>Post Order Traversal</strong>: Left, Right, Root. <br>In our example: H I D J E B F K G C&#xA0;A</li><li><strong>Level Order Traversal</strong>, also known as Breadth-first search. <br>In our example: A B C D E F G H I J&#xA0;K</li></ul><p>In that tutorial, you are going to learn how to implement these different Tree Traversal Algorithms in Java with recursion and without recursion.
